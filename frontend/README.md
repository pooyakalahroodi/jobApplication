# Job Applications Frontend

React dashboard for the local job applications backend.

## Local Setup

```powershell
cd frontend
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:5173
```

The backend should be running at:

```text
http://127.0.0.1:8000
```

Override the backend URL with:

```text
VITE_API_BASE_URL=http://127.0.0.1:8000
```
